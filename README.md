# Threading Histories: Nigerian Women's Textile Traditions

A digital humanities project mapping the spatial, temporal, and cultural dimensions of Nigerian women's textile handicraft traditions from 1850-1960.

## Project Description

This interactive website examines how British colonialism disrupted traditional knowledge systems and transformed women's labor practices in Nigerian textile production.

**Project URL:** threadinghistories.org (coming May 2025)

## CHI Fellowship

This project is being developed as part of the Cultural Heritage Informatics Fellowship at Michigan State University (January-May 2025).

## Technology Stack

- HTML5/CSS3/JavaScript
- Leaflet.js (interactive mapping)
- GeoJSON (location data)
- GitHub Pages (hosting)
